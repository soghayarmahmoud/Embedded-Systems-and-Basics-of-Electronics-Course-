# Electronic Components Guide

## What this topic covers
- resistors, capacitors, inductors, diodes, and transistors
- component symbols and practical functions
- component selection for simple circuits

## How to study it
1. Learn the purpose of each component.
2. Study how each one behaves in a circuit.
3. Practice identifying components on a breadboard.
4. Connect each component to a tiny robotics example.

## Suggested practice
- build a simple LED circuit
- test a resistor divider
- observe capacitor charging behavior

## Goal
Understand the basic building blocks used in every electronic system.
