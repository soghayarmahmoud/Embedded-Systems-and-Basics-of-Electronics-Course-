# Programming Fundamentals Guide

## What this topic covers
- variables, conditions, loops, and functions
- embedded C or Arduino-style programming
- debugging and code structure

## How to study it
1. Learn the basic syntax and structure of the language.
2. Practice small programs before using hardware.
3. Connect each concept to blinking LEDs, sensors, or motors.

## Suggested practice
- write a program that reads a sensor
- control an LED with a button
- make a simple state machine

## Goal
Write reliable code for embedded systems and robot behavior.
