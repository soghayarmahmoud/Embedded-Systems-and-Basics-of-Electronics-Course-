# Microcontrollers Guide

## What this topic covers
- microcontroller architecture
- memory, CPU, and peripherals
- GPIO, timers, interrupts, and ADC

## How to study it
1. Learn the basic architecture of a microcontroller.
2. Study how input and output pins work.
3. Practice with a simple board such as Arduino or STM32.

## Suggested practice
- blink an LED
- read a button input
- use PWM for motor speed control

## Goal
Be able to program and control a microcontroller for simple robot tasks.
