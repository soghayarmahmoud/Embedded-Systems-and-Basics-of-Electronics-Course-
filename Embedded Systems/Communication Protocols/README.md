# Communication Protocols Guide

## What this topic covers
- UART, SPI, I2C, and CAN
- data exchange between devices
- sensor and actuator communication

## How to study it
1. Learn when each protocol is used.
2. Compare speed, wiring, and complexity.
3. Practice with simple examples and hardware modules.

## Suggested practice
- connect a sensor using I2C
- send data using UART
- test SPI communication between modules

## Goal
Understand how embedded systems exchange data in real devices.
